# CC:Tweaked Server Wide Web Server and Client ENV

This is a "Browser" for Multiplayer/Singleplayer Server using Rednet and the basalt UI.
Its easy to use and people who want to make websites can easily be coded using minimal
lua understanding and the basalt ui coding solution!
# How to use the Client side?
install and run client.lua after that it`s just like a Browser.
# How to use the Server side?
install and open the file and configure the following values to your liking:
servername; domain
